# Dear Psy-Lovers,

Ever wished to reproduce that classic lead sound, bassline, but sound engineering left you hanging? This repo shall serve our fellow psy-heads by providing presets, so we can learn from each other and improve our sound designs and spread the love.

# How to contribute

Anyone may contribute.

Ways to contribute:
- submit files via Pull Request
- open Issues
- suggest ideas in Discussions

# Rules
- For each preset provided, please give a short description in the respective README.md file, being source youtube tutorials, soundtracks or a qualitative description.  
- no illegal content
- no hate speech
- no malware
- respect folder structure

# Future plans
- MIDI pattern repositories for basslines and leads
